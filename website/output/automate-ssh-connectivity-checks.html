automate SSH connectivity checks

Posted on: Mon 02 July 2018

Tags: #linux, #bash, #scripting, #automation

say you need to test ssh connectivity to possibly hundreds of servers, you can automate this using a while loop in bash.

1
2
3
4
5

#!/bin/bash
# SSH connectivity &amp;&amp; authentication checks

while read host; do
    echo "$host" &gt;&gt; ssh_output.txt ;  ssh -T -o ConnectTimeout=10 -o PasswordAuthentication=no $host 2&gt;&gt;ssh_output.txt
done &lt; servers.txt

note that we want to test two things: (1) connectivity and (2) whether authentication works or not. therefore simply port scanning will not be sufficent here.

obviously we can add more to this. for example, we could set this to alert in cron via mail or use sendmail to post an alert.
